# 中国居民收入分布参考（2024年）

## 项目简介

这是一个交互式的中国居民收入分布参考网页，帮助用户了解2024年中国居民收入的分布情况和统计数据。

## 功能特点

- 🔍 **收入查询**：快速查询不同收入水平的人群分布
- 📊 **数据可视化**：直观展示收入分布曲线
- 💰 **收入定位**：了解自己在人群中的收入位置
- 📱 **响应式设计**：支持桌面和移动设备访问
- 🎨 **美观界面**：现代化的UI设计和用户友好的交互体验

## 数据来源

本项目数据基于中国国家统计局官方发布的居民收入统计数据：

- **数据年份**：2024年
- **数据来源**：国家统计局（National Bureau of Statistics of China）
- **统计范围**：全国居民收入抽样调查数据

## 技术栈

- **HTML5**：页面结构
- **CSS3**：样式和响应式布局
- **JavaScript (ES6+)**：交互逻辑和数据处理
- **Canvas API**：数据可视化图表绘制
